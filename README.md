# Notes App

Небольшое React-приложение для создания и управления заметками.

## Функциональность
- добавление заметки
- удаление заметки
- выбор цвета для заметки
- переключение темы (светлая, тёмная, системная)
- подсчёт активных заметок
- сохранение заметок в localStorage

## Технологии
- React
- TypeScript
- Emotion (CSS-in-JS)
- Vite

## Запуск проекта
```bash
npm install
npm run dev
